PySitHere è un’applicazione desktop moderna per la gestione delle prenotazioni delle postazioni in ufficio.
Sviluppata in Python con interfaccia grafica CustomTkinter, permette agli utenti di prenotare scrivanie tramite una mini‑mappa interattiva, visualizzare la disponibilità giornaliera, settimanale o mensile, e gestire le proprie prenotazioni.
Gli amministratori possono configurare uffici e piani, modificare la disposizione delle scrivanie tramite editor grafico e gestire gli utenti del sistema.
La configurazione degli ambienti avviene tramite un semplice file JSON, che definisce uffici, piani e coordinate delle postazioni.



PySitHere is a modern desktop application for managing office desk reservations.
Built in Python with a CustomTkinter graphical interface, it allows users to book desks through an interactive mini‑map, view availability by day, week, or month, and manage their personal bookings.
Administrators can configure offices and floors, edit desk layouts using a graphical editor, and manage system users.
Office layouts are defined through a simple JSON configuration file that specifies offices, floors, and desk coordinates.
